ikdp2lat is a small R package which performs following tasks:

  • Transforming metadata from FileMaker Pro to CMDI
  • Performing general validations for the data
  • Preparing ZIP files to be uploaded to LAT

The basic motivation behind this packages is the idea that in order to add a session into TLA archive the only parameter one would truly need to specify is the session name. If the metadata and files are arranged correctly, the program will be able to resolve without assistance which files belong where.
